Simulação de Ameaças em Python

---
Sobre o Projeto

Este projeto tem como objetivo estudar, de forma simples e segura, como alguns tipos de ameaças funcionam na prática.

⚠️ Uso apenas educacional.

---
Ransomware (Simulação)

Foi estudado como um ransomware funciona:

->Altera (simulando criptografia)
->Exibe uma mensagem de resgate

Tudo feito com arquivos de teste, sem risco.

---
Keylogger (Simulação)

Foi analisado o funcionamento de um keylogger:

->Captura teclas digitadas
->Registra em um arquivo
->Funcionamento em segundo plano

Apenas estudo conceitual em ambiente controlado.

---
Como se proteger

->Usar antivírus
->Manter o sistema atualizado
->Evitar arquivos e links suspeitos
->Utilizar firewall
->Ter cuidado com e-mails desconhecidos

---
Aprendizado

->Entendi como ataques funcionam
->Percebi a importância da segurança
->Aprendi formas de prevenção
